# Step 6 — Multiply

Take a **validated winning ad** and multiply its strategic DNA across all 40 format templates. Output: up to 39 new variations that keep persona/angle/emotion/product truth intact but swap the structural packaging.

## Files

- [`multiply-prompt.md`](multiply-prompt.md) — the prompt + governance rules + output structure

## Prerequisites

- Validated winner from steps 1-5 (real performance signal, not just a scored brief)
- All 40 format templates in [`../04-format-templates-copy-prompt/01-templates-by-motion/`](../04-format-templates-copy-prompt/01-templates-by-motion/)
- Brand cards + product photo ready for upload

## Workflow in 1 line

Winner brief → multiply prompt → 39 variations → score each through agents → generate prompts → run in market → identify which format packagings amplify the winner's DNA best for this persona.
