# 08 — Before & After (UGC Native)

🏷️ Mirror selfie transformation. Must look like a real person posted it.

## Metadata

- **aspect_ratio:** 9:16 default (1080x1920)
- **needs_product_images:** false (product color reference only)
- **reference_uploads:** product image (for brand color tone reference ONLY — NOT as a hero element)
- **top_safe_zone:** 270px
- **bottom_safe_zone:** 340px

## Product reference rule (CRITICAL)

This format does NOT render the product as hero. Product reference is used only for brand-color tone. The image prompt should say: *"Use the attached product image for brand color reference only. The product may appear subtly on the counter in the AFTER frame, reproduced exactly per reference, but is not the visual hero."*

## What this format is

Native UGC before/after transformation. Side-by-side (or top/bottom) selfie frames shot in the same bathroom — left/top is grainy, dim, "before" state; right/bottom is bright, natural, "after" state. Handwritten-style date labels. Looks stitched together in CapCut, not professionally designed. The format wins by NOT looking like an ad.

## Copy template

```
[BEFORE DATE]    - Short date or day label. e.g., "DAY 1" / "March 3" / "Week 1"
                   Handwritten-style white text.

[AFTER DATE]     - Corresponding label. e.g., "DAY 30" / "April 3" / "Week 5"

[TIMEFRAME TAG]  - Top center. "[TIMEFRAME] on [BRAND]"
                   e.g., "30 days on [brand]" with a small emoji.

No headlines. No CTA. No studio branding. The transformation is the copy.
```

## Image generation prompt

```
Use the attached product image for brand color reference only. This should look like a real person's post, NOT a polished ad.

Create a TikTok-style before-and-after UGC transformation. [ASPECT RATIO — 1080x1920, 9:16 vertical].

COMPOSITION:
One continuous image filling the entire frame, split horizontally into LEFT and RIGHT halves (or top/bottom per variation).

LEFT: grainy iPhone mirror selfie. [PERSON DESCRIPTION — age, hair, clothing from persona spec] standing in a dimly lit bathroom, holding phone casually (mirror selfie or self-portrait), [BEFORE STATE — visible concern the product addresses]. Harsh overhead bathroom lighting. Slightly grainy quality consistent with a 2-year-old iPhone. Natural, unposed.

RIGHT: same person, same bathroom, but with bright natural window light. [AFTER STATE — visible improvement related to the product's claim]. [PRODUCT] visible casually on the counter behind them or in the bottom corner, reproduced exactly from reference image but subtle — not hero.

Top center: white handwritten-style text overlay "[TIMEFRAME TAG]" with a small [EMOJI].

LEFT side: white handwritten-style date text "[BEFORE DATE]" in the bottom corner of the left half.
RIGHT side: white handwritten-style date text "[AFTER DATE]" in the bottom corner of the right half.

TYPOGRAPHY:
Handwritten-style or casual sans-serif for date labels (e.g., TikTok's default text tool look). Not premium typography — this should feel like it was added in CapCut.

SAFE ZONES:
Top [TOP SAFE ZONE]px and bottom [BOTTOM SAFE ZONE]px clear.

COLOR AND TREATMENT:
LEFT: dim, slightly cool, slightly grainy. RIGHT: bright, warm, natural-looking. The lighting contrast is the transformation.

BRAND IDENTITY:
NO brand logo overlay. Brand identity comes ONLY from the product visible on the counter in the AFTER frame.

MOOD:
Authentic, earned, raw. "I can't believe the difference."
```

## Variation vectors

**Split direction**: left/right side-by-side (default) | top/bottom stacked | diagonal split

**Setting**: bathroom mirror (default) | bedroom mirror | car visor mirror | gym changing room

**Transformation type**: visible skin/hair/body change | energy/mood shift (harder to show visually) | environment change (messy→organized for lifestyle products)

**Text placement**: date labels in corners (default) | timeframe tag only on top | running caption along bottom

**Emoji use**: one small emoji on top timeframe tag | emojis on both dates | no emoji (drier)

**Grain level**: subtle (default) | heavy (early-iPhone vibe) | clean (softer UGC)

## Format-specific rules

- **It must NOT look like an ad.** If the compositing looks designed, the whole effect collapses. No gradients, no brand headlines, no polish.
- **Product is present, not promoted.** The product appears subtly on the counter in the AFTER frame — reproduced exactly per the reference image — but it's not hero-lit, not centered, not labeled with an arrow.
- **Honest transformation.** The "before" and "after" should be plausibly the same person photographed at different times. Don't fabricate impossible changes.
- **Agent 4 (Brand Consistency) may down-score this format if the brand's visual identity is premium/polished.** UGC native works for accessible / community brands, not for luxury. Check brand spec before using.
- **Claims compliance.** Any visible change implies a claim. The brand's regulatory / claims team must approve what transformation is shown before production.
